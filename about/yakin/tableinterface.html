Interface AUTH/Login pada E-Commerce

INTERFACE KETERANGAN
Login user pada interface ini user login menggunakan email dan password untuk masuk ke halaman beranda atau dashboard utaman pada E-Commerce.
Login Admin pada interface ini, Admin dapat login menggunakan email dan juga password juga untuk masuk pada halaman beranda admin atau dashboard utama admin, bedanya dengan halaman login user adalah pada login admin ini cuma login saja tidak ada lupa password atau buat akun.
Lupa Password pada interface ini, ketika user lupa dengan passwordnya akunnya dapat dipulihkan menggunakan email aktif dimana user dapat memperbarui password ketika sudah melakukan verifikasi email.
Daftar pada interface ini, ketika User belum mempunyai akun, user daftar terlebih dahulu dengan mengisi biodata yang tertera pada halaman daftar.
Verifikasi Email pada interface ini user memasukkan email pemulihan supaya dapat mereset sandi atau membuat sandi baru ketika lupa password.
Buat Sandi Baru pada interface ini, User membuat sandi baru sesudah verivikasi email akun pemulihan.
